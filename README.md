Проект который использует python GUI интерфейс для того, чтобы редактировать необходимые моменты в Advanced Installer проекте

Позволяет:
1. Менять названия и имена приложения внутри проекта
2. Принимать .png/.jpg файл и конвертировать его в .ico + .bmp для использования вместо текущих иконок в проекте
3. Запускать build проекта(для этого требуется передать приложению пути к Advanced Installer если оно не нашло установку автоматически) 
4. Редактировать файл Invoke-AppDeployToolkit.ps1 - для каждого проекта отдельно(создавать версии)
